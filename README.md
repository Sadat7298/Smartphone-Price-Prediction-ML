# Smartphone Price Prediction using Machine Learning

## Project Overview

This project focuses on predicting smartphone prices using Machine Learning techniques. It analyzes smartphone specifications, user information, and customer ratings to identify the factors that influence smartphone pricing.

---

## Problem Statement

The smartphone market is growing rapidly, with new devices launched frequently. Determining the appropriate price of a smartphone based on its specifications is an important task for manufacturers, retailers, and customers.

The aim of this project is to build a Machine Learning model that can accurately predict smartphone prices using historical smartphone data.

---

## Objective
-Analyze smartphone datasets to understand pricing trends.
-Clean and Preprocess the data for analysis.
-Merge multiple datasets into a single dataset.
-Perform Exploratory Data Analysis(EDA).
-Train and evaluate a machine learning model for smartphone price prediction.
-Generate accurate price predictions based on smartphone features.

---
## Dataset
This project uses three datasets:
-cellphone_users.xlsx - User Information.
-cellphones_data.xlsx - Smartphone specifications and details.
-cellphones_ratings.xlsx - User ratings and reviews.

## Project Workflow

## Step 1- Data Collection
Import all three datasets into Jupyter Notebook.

## Step2- Data Cleaning
-Remove duplicate records
-Handle missing values.
-Correct inconsistent data.

## Step 3- Data Preprocessing
-Merge the Datasets.
-Convert categorical variables into numerical values.
-Prepare the dataset for machine learning.

## Step 4- Exploratory Data Analysis(EDA)
Analyze the dataset using visualizations such as:
-Distribution plots
-Correlation heatmap
-Count plots.
-Box plots
-Histograms

## Step 6- Model Building
Train the machine learning model using the processed dataset.

## Step 7- Model Evaluation
-Mean Absolute Error(MAE)
-Mean Squared Error(MSE)
-Root Mean Squared Error(RMSE)
-R^2 Score
